   QwikiA 
-----------
version 1.1.1

QwikiA is a question-and-answer website where questions are asked, answered, edited and organized by its community of users.

Requirements:
- Server
- PHP 5.4+
- MySql 5.x
- ModRewrite enabled
- PDO library enabled
- GD library enabled
- Valid SMTP server for you to use email

Installation:


1) Installation in root directory:
- Copy all files to your server.
- Open website url and start installation.
- After installation - rename install folder.
- After installation you can add demo content. You can download it from here: https://www.dropbox.com/s/ty21dsjacgtvi9o/demo_content.zip

2) Installation in subfolder:
- Copy all files to your server folder.
- Open .htaccess file and add on line 4 subfolder name after RewriteBase/
- Open website url and start installation.
- After installation - rename install folder.
- After installation you can add demo content. You can download it from here: https://www.dropbox.com/s/ty21dsjacgtvi9o/demo_content.zip
